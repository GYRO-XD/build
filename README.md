# 🚀 PYSOBuilder - Python to .so Compiler

**Professional Python to .cpython-310.so Compiler Tool**

[![Version](https://img.shields.io/badge/version-1.0.0-blue)](https://github.com/GYRO-XD/pyso-builder)
[![Python](https://img.shields.io/badge/python-3.8+-green)](https://www.python.org/)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

## ✨ Features

- 🔥 **One Command Compilation** - Compile Python to .so instantly
- 🚀 **Optimization Levels** - 0 to 3 for performance tuning
- 📦 **Batch Compilation** - Compile multiple files at once
- 💻 **Platform Detection** - Auto-detect and name accordingly
- 🧪 **Auto-Import Testing** - Verify compilation works
- 📊 **JSON Reports** - Generate compilation reports
- 🧹 **Automatic Cleanup** - Remove temporary files
- 🔧 **Verbose Mode** - Detailed compilation output

## 📦 Installation

### Quick Install
```bash
git clone https://github.com/GYRO-XD/build.git
cd build
chmod +x builder.sh
./builder.sh
```


### Clone the repository
git clone https://github.com/GYRO-XD/build.git
cd build

### Install dependencies
pip install -r requirements.txt

### Use the tool
python pyso_builder.py your_script.py
